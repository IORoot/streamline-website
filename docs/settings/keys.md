# Keyboard Shortcuts

The Keys settings tab displays all available keyboard shortcuts and allows you to configure canvas interaction preferences.

## Keyboard Shortcuts Reference

### Node Operations

| Action | Shortcut |
|--------|----------|
| Delete selected node(s) | `Delete` |
| Toggle minimize/maximize all nodes | `M` |

### Pipeline Operations

| Action | Shortcut |
|--------|----------|
| Show preview modal | `P` |
| Run pipeline | `R` |
| Clear canvas | `C` |

### Zoom Operations

| Action | Shortcut |
|--------|----------|
| Zoom in | `+` |
| Zoom out | `-` |
| Reset zoom to fit all nodes | `0` |

## Canvas Controls

### Canvas Drag Button

Configure which mouse button is used to drag the canvas:

- **Left-Click**: Use the left mouse button to drag the canvas
- **Middle-Click**: Use the middle mouse button (scroll wheel click) to drag the canvas
- **Right-Click**: Use the right mouse button to drag the canvas

**Note**: Changing this setting may affect how you interact with nodes and context menus. Choose the option that works best for your workflow.

## Important Notes

- Keyboard shortcuts won't trigger when typing in text fields or text areas
- Shortcuts are case-insensitive
- Some shortcuts may conflict with browser or system shortcuts
- The canvas drag button setting is saved automatically

## Usage Tips

- Use `M` to quickly minimize all nodes for a cleaner view
- Use `0` to quickly fit all nodes in view
- Use `R` to run your pipeline without clicking the toolbar button
- Use `P` to quickly preview your pipeline output
- Configure the canvas drag button to avoid conflicts with node selection
