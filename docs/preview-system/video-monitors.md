# Video Monitors

The Preview Modal includes professional video analysis tools: waveform monitor, vectorscope, histogram, and parade displays to help you analyze and correct video color and exposure.

## Waveform Monitor

The waveform monitor displays the luminance (brightness) levels of your video image.

### Features

- **Luminance display**: Shows brightness levels across the image
- **Horizontal axis**: Represents horizontal position in the image
- **Vertical axis**: Represents brightness levels (0-100 IRE or 0-255)
- **Real-time updates**: Updates as you change nodes or parameters

### Using the Waveform Monitor

1. Open the Preview Modal
2. Select a video/image node
3. Click the "Waveform Monitor" button in the toolbar
4. The waveform display appears over the preview

### Interpreting the Waveform

- **High values**: Bright areas of the image
- **Low values**: Dark areas of the image
- **Flat lines**: Uniform brightness in that area
- **Peaks**: Bright highlights
- **Valleys**: Dark shadows

### Applications

- **Exposure correction**: Identify overexposed or underexposed areas
- **Contrast analysis**: See the distribution of brightness levels
- **Color grading**: Understand luminance before color correction
- **Quality control**: Ensure proper exposure levels

## Histogram

The histogram displays the distribution of pixel values across the image.

### Features

- **Brightness distribution**: Shows how many pixels are at each brightness level
- **Color channels**: Can display RGB channels separately
- **Statistical analysis**: Helps understand image exposure
- **Real-time updates**: Updates as you modify nodes

### Using the Histogram

1. Open the Preview Modal
2. Select a video/image node
3. Click the "Histogram" button in the toolbar
4. The histogram display appears

### Interpreting the Histogram

- **Left side**: Dark pixels (shadows)
- **Right side**: Bright pixels (highlights)
- **Peaks**: Common brightness levels
- **Gaps**: Missing brightness levels
- **Clipping**: Peaks at the edges indicate over/underexposure

### Applications

- **Exposure analysis**: Identify exposure problems
- **Contrast adjustment**: Understand brightness distribution
- **Color correction**: Analyze individual color channels
- **Quality control**: Ensure proper exposure range

## Vectorscope

The vectorscope displays color information in a circular graph, showing hue and saturation.

### Features

- **Hue display**: Angle represents color hue
- **Saturation display**: Distance from center represents saturation
- **Color targets**: Reference targets for skin tones and primary colors
- **Real-time updates**: Updates as you modify color parameters

### Using the Vectorscope

1. Open the Preview Modal
2. Select a video/image node
3. Click the "Vectorscope" button in the toolbar
4. The vectorscope display appears

### Interpreting the Vectorscope

- **Center**: Neutral colors (grayscale)
- **Outer edge**: Highly saturated colors
- **Angles**: Represent different hues
- **Targets**: Reference points for skin tones and primaries

### Applications

- **Color correction**: Adjust hue and saturation precisely
- **Skin tone correction**: Use targets to maintain natural skin tones
- **Color matching**: Match colors between shots
- **Creative grading**: Achieve specific color looks

## Parade Display

The parade display shows RGB channels side-by-side for detailed color analysis.

### Features

- **RGB separation**: Shows red, green, and blue channels separately
- **Channel comparison**: Compare channel levels easily
- **Exposure analysis**: Identify channel-specific exposure issues
- **Real-time updates**: Updates as you modify nodes

### Using the Parade Display

1. Open the Preview Modal
2. Select a video/image node
3. Click the "Parade" button in the toolbar
4. The parade display appears

### Interpreting the Parade

- **Three displays**: Red, Green, Blue channels
- **Height**: Represents brightness of that channel
- **Alignment**: Channels should align for neutral colors
- **Imbalance**: Misalignment indicates color cast

### Applications

- **Color balance**: Identify and correct color casts
- **Channel analysis**: Understand individual channel contributions
- **White balance**: Ensure proper white balance
- **Color correction**: Make precise color adjustments

## Spectrum Analyzer

The spectrum analyzer displays frequency information for audio content in video.

### Features

- **Frequency display**: Shows frequency content over time
- **Audio analysis**: Analyze audio characteristics
- **Real-time updates**: Updates during playback

### Using the Spectrum Analyzer

1. Open the Preview Modal
2. Select a node with audio
3. Click the "Spectrum" button in the toolbar
4. The spectrum display appears

## Switching Between Monitors

- Click monitor buttons in the toolbar to switch displays
- Only one monitor can be active at a time
- Click the active monitor button again to hide it
- Monitors overlay the preview image

## Best Practices

### Exposure Correction

1. Use waveform monitor to identify exposure issues
2. Use histogram to understand brightness distribution
3. Adjust nodes to correct exposure
4. Verify with waveform and histogram

### Color Correction

1. Use vectorscope to understand color relationships
2. Use parade to identify color casts
3. Make color adjustments
4. Verify with vectorscope and parade

### Quality Control

1. Check waveform for proper exposure range
2. Verify histogram for good distribution
3. Use vectorscope to check color accuracy
4. Use parade to ensure color balance

## Tips

- Use multiple monitors together for comprehensive analysis
- Waveform and histogram complement each other
- Vectorscope is essential for color work
- Parade helps identify subtle color issues
- Monitors update in real-time as you adjust nodes
