# Installation

STREAMLINE is available as a desktop application for macOS, Windows, and Linux.

## System Requirements

- **macOS**: 10.15 or later
- **Windows**: Windows 10 or later
- **Linux**: Ubuntu 18.04 or later (or equivalent)

## Download

Download the latest version of STREAMLINE from the [GitHub Releases](https://github.com/IORoot/streamline/releases) page.

## Installation Steps

### macOS

1. Download the `.dmg` file
2. Open the downloaded file
3. Drag STREAMLINE to your Applications folder
4. Open STREAMLINE from Applications

### Windows

1. Download the `.exe` installer
2. Run the installer
3. Follow the installation wizard
4. Launch STREAMLINE from the Start menu

### Linux

1. Download the `.AppImage` file
2. Make it executable: `chmod +x Streamline-*.AppImage`
3. Run: `./Streamline-*.AppImage`

## First Launch

When you first launch STREAMLINE, you'll see the welcome screen with options to:
- Create a new pipeline
- Open an existing pipeline
- View example pipelines
- Access the documentation

## Next Steps

Now that you have STREAMLINE installed, check out the [Quick Start Guide](/docs/getting-started/quick-start) to create your first pipeline.
